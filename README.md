# Water-Tracker
Application Name: Water Tracker

Team Members: Landon Fluckiger

Project Goals: Create an app that is both quick and easy to use in order to take up as little time as possible. The goal is to allow people to track their water without it becoming a burden. This will help hold people accountable to drinking enough water each day.

Features: The application has 3 screens. The first screen is where the user can quick add water (with preset water amounts to add), along with a bar that they can use to enter a custom amount. The second screen can keep track of the user's streak. It can desplay how many days in a row the user has met their water intake goal. The third screen allows the user to set their water intake goal.

Design: The design is as simplistic as possible. We want the user to think as little as they can about having to input water intake, while also holding them accountable to their water intake. It will be split into 3 screens to make it clear what each page does.
